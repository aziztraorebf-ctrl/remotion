## Notes Techniques

### Duree estimee par acte
| Acte | Duree | Mots | Frames (30fps) |

| Hook | 20s | 50 | 600 |
| Setup | 60s | 150 | 1800 |

| Reveal + CTA | 70s | 200 | 2100 |
| **TOTAL** | **240s (4 min)** | **~600** | **7200** |

### Composants Remotion necessaires (Brutalist Hybride)
- **BrutalistBackground** : fond flat par acte (couleur unie, pas de degradees)
- **StampText** : texte opacity 0->1 instantane
- **AnimatedCounter** : spring rapide (damping:20, stiffness:180)
- **BrutalistBar** : barre horizontale, 0 border-radius, bordure 4px
- **SplitBar** : barre qui se divise en 2 portions (capital vs interets)
- **LineChart** : trait 4px, points carres 8x8, draw progressif
- **ComparisonBlocks** : deux rectangles cote a cote, bordures 4px
- **GlitchTransition** : decalage RGB 2-3 frames entre actes
- **StickFigure** : cercle + rectangle minimal (Thomas)
- **HouseDouble** : maison geometrique qui se dedouble (bien + fantome)
- **StrategyPills** : tags rectangulaires stamps sequentiels

### Voice markers (ElevenLabs Chris)
- MAJUSCULES = emphasis fort
- "..." = pause 0.3-0.5s
- "!" = energie montante
- "--" = break de ton
- "?" = inflexion interrogative

### Polices
- Titres/stamps : Space Grotesk Bold, MAJUSCULES
- Chiffres : JetBrains Mono Bold, tabular
- Corps : Space Grotesk Regular
- Labels : JetBrains Mono Regular, MAJUSCULES
