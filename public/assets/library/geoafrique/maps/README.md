# GeoAfrique - Map Assets

## Style: Modern Geopolitical Briefing

### Congo-Brazzaville Map
- **Source**: ChatGPT Image (GPT-4o)
- **File**: `congo-brazzaville-geopolitical-ref.png`
- **Style**: Modern geopolitical infographic, glowing gold country on dark background
- **Background**: Very dark charcoal with subtle grid lines
- **Ocean**: Dark navy, no decorative elements
- **Text**: NONE (all labels added via Remotion)

**Prompt used**:
> Map of Congo-Brazzaville highlighted in bright glowing gold on a dark map of Central Africa. Style: modern geopolitical infographic, clean vector look, dark charcoal background. Surrounding countries in dark gray with subtle borders. Atlantic ocean in very dark navy, no decorative elements. Subtle grid lines overlay like a military intelligence briefing map. No text, no labels, no country names, no city markers, no annotations of any kind. Dramatic lighting on the gold country. Square format, 4K quality.

### Usage in Remotion
- Glow pulse: animated brightness + drop-shadow on the gold country
- Reveal: radial mask animation from center outward
- Ken Burns: wide Africa view zooming into Congo
- Gold particles floating around country border
- Transition: map zoom into Brazzaville -> crossfade to portrait

### Design Rules
- ALL text/labels added in Remotion code (never baked into images)
- Images are canvases, Remotion is the typographer
- Sans-serif white font for country labels
- Animated arrows and markers via React components
