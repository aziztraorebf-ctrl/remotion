# Plan : Hook Bloc 1 - Peste 1347 (0:00 - 0:50)

## Goal
Implementer les 50 premieres secondes de la video Peste 1347 : 7 sous-scenes Hook avec carte animee de propagation de la peste (scenes 1-4), puis transition vers le monde pixel medieval (scenes 5-7) pour le reframe "HUMAINS, pas la maladie".

## Architecture

```
HookScene.tsx (orchestrateur)
  |-- PlagueSpreadMap.tsx (scenes 1-5) -- carte Eurasie animee d3-geo
  |-- GraveyardBackground.tsx (scenes 6-7) -- parallax CraftPix layers
  |-- HookTextOverlay.tsx (toutes scenes) -- stamp/strikethrough/typewriter
  |-- CRTOverlay (existant) -- couche permanente
  |-- GridSpritesheetAnimator (existant) -- silhouettes scenes 6-7
  |-- PixelTypewriter (existant) -- labels carte
```

6 fichiers a creer, 2 fichiers config a modifier, 5 composants existants reutilises.

## Tech Stack
- Remotion (React, `Sequence`, `spring()`, `interpolate()`, `useCurrentFrame()`)
- d3-geo + topojson-client (deja installes)
- FPS=30, viewport 1920x1080
- Fonts: PressStart2P (titres) + VT323 (body) via @remotion/google-fonts

### 1a. hookTiming.ts

```ts
// src/projects/peste-1347-pixel/config/hookTiming.ts
import { FPS } from "./timing";

// Hook scene durations in seconds (PROVISIONAL - will be recalibrated after ElevenLabs audio)
const HOOK_DURATIONS_SEC = [8, 10, 8, 3, 3, 6, 5] as const;

// Convert to frames
const HOOK_DURATIONS = HOOK_DURATIONS_SEC.map((s) => s * FPS) as unknown as [
  number, number, number, number, number, number, number
];

// Cumulative start frames for each scene
export const HOOK_SCENE_STARTS = HOOK_DURATIONS.reduce<number[]>(
  (acc, dur, i) => {
    if (i === 0) return [0];
    return [...acc, acc[i - 1] + HOOK_DURATIONS[i - 1]];
  },
  []
) as [number, number, number, number, number, number, number];

export const HOOK_SCENE_DURATIONS = HOOK_DURATIONS;

export const HOOK_TOTAL_FRAMES = HOOK_DURATIONS.reduce((a, b) => a + b, 0);

// Scene indices for readability
export const SCENE = {
  ISSYK_KUL: 0,
  CATAPULTE: 1,
  GALERES: 2,
  MOITIE: 3,
  REFRAME: 4,
  REVEAL: 5,
  REFLEXES: 6,
} as const;
```

## Task 2 : PlagueSpreadMap.tsx (carte animee Eurasie)

**But** : Copier FilledEuropeMap.tsx et l'etendre pour couvrir l'Eurasie (Issyk-Kul a l'est), avec camera animee, routes commerciales, et mode "invasion" cascade.

### Modifications par rapport a FilledEuropeMap

1. **Projection etendue** : Centre passe de `[15, 52]` a `[40, 45]` et scale de `800` a `500` pour couvrir l'Asie Centrale
2. **Camera animee** : Systeme d'interpolation camera avec 4 etats :
   - Scene 1 : Zoom sur Issyk-Kul (center [77, 42.5], scale 1200)
   - Scene 2 : Pan vers Caffa (center [55, 43], scale 800)
   - Scene 3 : Dezoom Mediterranee (center [25, 42], scale 600)
   - Scene 4 : Vue complete Europe (center [15, 50], scale 500)
3. **Routes animees** : Trait dore anime le long des `HOOK_ROUTES` avec `interpolate()` sur `strokeDashoffset`
4. **Galeres SVG** : 3-4 sprites naviguant le long de la route maritime (scene 3)
5. **Mode invasion** : Scene 4, cascade a 2 vitesses -- villes majeures d'abord (Messine, Marseille, Paris, Londres) avec 4-5 frames d'ecart, puis les secondaires en burst rapide (2 frames). Le texte "MOITIE" arrive APRES la cascade, pas pendant.

### Elements cles a implementer

**Fichier** : Creer `src/projects/peste-1347-pixel/components/GraveyardBackground.tsx`

### Assets disponibles

- `public/assets/peste-pixel/craftpix/graveyard/background_4/` : 7 layers (576x324 natif)
  - Layer 1 = ciel le plus profond, Layer 7 = premier plan
- Utiliser `background_4` (7 layers = meilleur parallax)

### Implementation

```ts
interface GraveyardBackgroundProps {
  localFrame: number;
  fadeInDuration?: number; // frames for crossfade entrance
  kenBurnsZoom?: number;  // target zoom (e.g. 1.02)
}
```

### Elements cles

- 7 `<Img>` Remotion empiles, chaque layer a un facteur parallax different (layer 1 = 0.02, layer 7 = 0.15)
- Parallax horizontal leger via `translateX(frame * factor)` et/ou vertical via `translateY`
- Ken Burns : `scale` interpole de 1.0 a `kenBurnsZoom` sur la duree de la scene. Valeur par defaut 1.05 (pas 1.02 -- trop subtil sur pixel art, risque shimmer). Si le parallax 7 layers suffit pour le mouvement, desactiver le Ken Burns.
- `imageRendering: "pixelated"` pour garder l'esthetique pixel
- Scale up de 576x324 a 1920x1080 (facteur ~3.33x)
- **Transition entree** : pas de crossfade depuis la carte (risque "milk" grisatre). Utiliser 2 frames de noir pur comme cut sec avant le graveyard.

## Task 4 : HookTextOverlay.tsx (textes multi-modes)

**But** : Creer un composant d'overlay texte avec 3 modes visuels pour les differents moments du Hook.

**Fichier** : Creer `src/projects/peste-1347-pixel/components/HookTextOverlay.tsx`

### 3 modes

1. **stamp** : Mot geant apparait avec spring() + leger shake (pour "MOITIE", "HUMAINS")
2. **strikethrough** : Mot apparait puis se barre avec une ligne rouge animee (pour "MALADIE")
3. **typewriter** : Texte apparait lettre par lettre (pour "LES MEMES REFLEXES")

### Props

```ts
interface HookTextOverlayProps {
  text: string;
  mode: "stamp" | "strikethrough" | "typewriter";
  startFrame: number;
  color?: string;
  fontSize?: number;
  position?: "center" | "bottom-center";
}
```

### Elements cles

- **stamp** : `spring({ frame, fps: 30, config: { mass: 2, damping: 12 } })` pour scale 0 -> 1, `opacity` 0 -> 1 en parallele
- **strikethrough** : texte normal + `<div>` rouge par-dessus, `width` interpole de 0% a 105% sur 10 frames
- **typewriter** : reutiliser la logique de `PixelTypewriter` (charsPerFrame + cursor clignotant)
- Font : PressStart2P pour stamp, VT323 pour typewriter
- Toujours centrer horizontalement, position verticale configurable

## Task 5 : HookScene.tsx (orchestrateur)

**But** : Assembler les 7 scenes du Hook dans un orchestrateur principal avec `Sequence`.

**Fichier** : Creer `src/projects/peste-1347-pixel/scenes/HookScene.tsx`

### Structure

```tsx
export const HookScene: React.FC = () => {
  const frame = useCurrentFrame();

  return (
    <AbsoluteFill style={{ backgroundColor: COLORS.bgAct1 }}>
      {/* Scene 1-5 : Carte animee (rendue UNE fois, pilotee par frame global) */}
      <Sequence from={HOOK_SCENE_STARTS[0]} durationInFrames={sumFrames(0, 4)}>
        <PlagueSpreadMap sceneIndex={getSceneIndex(frame)} localFrame={...} />
      </Sequence>

      {/* Scene 4 : Texte MOITIE stamp */}
      <Sequence from={HOOK_SCENE_STARTS[3]} durationInFrames={HOOK_SCENE_DURATIONS[3]}>
        <HookTextOverlay text="MOITIE" mode="stamp" startFrame={0} color={COLORS.plagueRed} fontSize={72} />
      </Sequence>

      {/* Scene 5 : MALADIE strikethrough */}
      <Sequence from={HOOK_SCENE_STARTS[4]} durationInFrames={HOOK_SCENE_DURATIONS[4]}>
        <HookTextOverlay text="MALADIE" mode="strikethrough" startFrame={0} color={COLORS.textPrimary} />
      </Sequence>

      {/* Scene 6 : Graveyard + silhouettes + HUMAINS */}
      <Sequence from={HOOK_SCENE_STARTS[5]} durationInFrames={HOOK_SCENE_DURATIONS[5]}>
        <GraveyardBackground localFrame={...} fadeInDuration={15} kenBurnsZoom={1.02} />
        {/* 3-4 silhouettes Gothicvania */}
        {/* HookTextOverlay "HUMAINS" stamp gold */}
      </Sequence>

      {/* Scene 7 : Overlay sombre + reflexes typewriter */}
      <Sequence from={HOOK_SCENE_STARTS[6]} durationInFrames={HOOK_SCENE_DURATIONS[6]}>
        <GraveyardBackground localFrame={...} />
        {/* Silhouettes marchent vers les bords */}
        {/* Overlay noir progressif */}
        <HookTextOverlay text="LES MEMES REFLEXES" mode="typewriter" startFrame={15} />
        {/* Cranes en constellation : pulse lie au rythme du typewriter (chaque nouveau mot = pulse) */}
      </Sequence>

      {/* CRT permanent -- scanlineOpacity reduit a 0.12 pendant scenes de texte pur (5, 7) */}
      <CRTOverlay />
    </AbsoluteFill>
  );
};
```

Appliquer `filter: brightness(0) contrast(100)` pour silhouettes noir pur (brightness(0) seul = gris sur pixels avec antialiasing).
